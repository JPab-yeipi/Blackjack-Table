<h1 align="center" style="color:#eb2f32;">♦️ Blackjack Table ♠️</h1>

Welcome to **Blackjack Table**, a full-stack implementation of the classic casino game. 
This project combines a robust, logic-driven **Python backend** (built 100% from scratch 
without AI assistance) with a modern, reactive **frontend** for a seamless 
and visual gaming experience.

---

<h2 align="center" style="color:#eb2f32;">🚀 Purpose</h2>

This project was built to strengthen my Object-Oriented Programming (OOP) skills by creating a complete Blackjack engine in Python from scratch. I wanted to focus on building a clean and scalable backend while also learning how to connect it to a modern frontend for a smooth and interactive full-stack experience.

More than just recreating Blackjack, this project became a fun way to practice software architecture, backend/frontend communication, and cleaner code design.
